# Pixifact Editor UI Prototype

这是一个独立的静态 UI 原型，用来重新确认 Pixifact AI-first 编辑器的信息架构和视觉方向。

它不实现任何业务逻辑，不读取真实项目，也不复用当前 `apps/editor` 的 React 代码。

## 打开方式

直接在浏览器打开：

```txt
apps/editor-ui-prototype/index.html
```

## 当前原型包含

- 文件系统面板
- UI 资源 / 节点树面板
- 中央画布
- AI 对话框
- Inspector 面板

## 目标

先确认布局、密度、信息层级和面板关系，再逐步把确认后的设计迁回真实编辑器。
