# pixif interactive showcase

This example demonstrates the main pixif primitives in one scene:

- `Application` and the root `Group`
- `GameObject.instantiate`
- `Component` updates driven by the application ticker
- `Layout`, `GridLayout`, `FlexGroup`, and `Flex`
- Pixi pointer events on pixif display objects
- DOM-backed `Input` and `Textarea`

Run the example from the repository root:

```bash
pnpm example
```

Build it with:

```bash
pnpm example:build
```
