# Pixifact v0.1.3

Pixifact 的首个 npm 发布版本。

## 已发布包

- `pixifact@0.1.3`
- `pixifact-cli@0.1.3`
- `create-pixifact@0.1.3`

## 安装

创建新的 Pixifact 项目：

```bash
bun create pixifact my-game
cd my-game
bun install
bun run build
```

在已有 Bun 项目中使用 Pixifact：

```bash
bun add pixifact pixi.js
bun add -d pixifact-cli
```

## 亮点

- `pixifact` 提供 runtime 扩展、项目配置和 compiler API。
- `pixifact-cli` 提供 Scene 校验和编译命令。
- `create-pixifact` 可生成最小 Bun + Vite + Pixifact 项目。
- `.scene` 文件仍是 AI-assisted Scene authoring 的 source of truth。

## 验证

发布后已通过以下命令验证 npm registry 版本：

```bash
npm view pixifact version --registry https://registry.npmjs.org/
npm view pixifact-cli version --registry https://registry.npmjs.org/
npm view create-pixifact version --registry https://registry.npmjs.org/
```

并通过真实 npm 安装路径验证：

```bash
bun create pixifact npm-smoke
cd npm-smoke
bun install
bun run build
```

smoke project 安装了 `pixifact@0.1.3` 和 `pixifact-cli@0.1.3`，成功编译 Scenes，并完成 Vite production build。

## 说明

- `pixifact-cli` 和 `create-pixifact` 在本版本中是 Bun-first 工具。
- 桌面 Editor 仍从源码仓库分发，不作为 npm 包发布。
- 仓库 Codex skills 仍从源码安装，后续再规划独立 npm skills 包。
